# Review Command

Perform code review without making changes.

## Instructions

1. Ask: "What would you like me to review?"

   - Specific files
   - Recent changes
   - Pull request

2. Review for:

   - **Bugs**: Logic errors, edge cases
   - **Performance**: Inefficient patterns, unnecessary operations
   - **Security**: Input validation, auth checks
   - **Style**: Code conventions, readability
   - **Tests**: Coverage, quality
   - **Documentation**: Missing or outdated docs
